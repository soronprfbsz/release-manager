## Infraeye2 표준본 설치

- 설명
    - 설치에 사용한 설치 파일 정보
      - 설치 파일
        - InfraEye-2.0.0.241127-STD.tar.gz
      - 도커 이미지 파일
        - docker pull 10.230.1.17:20081/ts/infraeye:2.0  
    - Infraeye2 설치가이드 문서에 기반하여 설치를 진행합니다.
- check points
    - 설치 전 infraeye 계정 수동 생성 필요        
        ```bash
        # 1. 사용자 계정 및 홈 디렉터리 생성
        sudo useradd -m infraeye
        
        # 2. 비밀번호 설정
        sudo passwd infraeye
        ```
        
    - 설치 전 MAC Address 정보 확인
        - MAC Address의 경우 설치 스크립트에서 자동 감지 해주지만 간혹 잘못 설정 되는 경우가 있으니 실제 host의 MAC Address가 맞는지 확인 필요
